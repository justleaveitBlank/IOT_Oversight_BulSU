extensionPROJECT ASSEMBLY
NOTE:

If you can’t broadcast your server on the internet, contact your Internet Service Provider (ISP) and ask them to enable Port-Forwarding on your Home Internet.

STEPS:
  1. Connect the Router to the modem and then plug it to a power outlet then turn it ON.
  2. Plug the Raspberry Pi into a power outlet and turn it ON.
  3. Plug the Power Socket Unit in a power outlet. The power socket unit should produce beep sound once the initialization completes.
  4. Please wait 2 to 3 minutes for the system to fully boot.
bookUser Guide
STEPS:
  1. Using your browser, go to the web address of the projects’ server that is being port forwarded (The web address depends on the port that is given to you by your ISP). Or you can also use the web mobile application that can be downloaded via Google drive Download here, after downloading the app, it will prompt for the project’s server address. In case of typographical errors, you can re-input the server address by tapping the home oversight logo 10 times or scan this QR CODE.
  2. Register for a user account.
  3. After user conformation for registration you now have access to the system. You can now register an appliance.
error_outlineTroubleshooting Guide

  • Power Socket Data wont update

    Make sure that the Router Access Point and raspberry pi/server pc is turned on and working properly. Restart the Power socket by doing a power cycle.

  • Cannot Access the server address

    Make sure that your local IP address is 192.168.2.119 and the router is connected to the modem that has access to the internet.

  • RFID won’t read when Appliance is plugged in

    Remove the plugged in appliance. Always put the RFID before plugging the device in.

  • Can't detect appliance plugged?

    Get your smart phone with camera and via camera see if the IR Collision Sensor have red or violet spot on.
